# Tech Domains Explorer

A beginner-friendly static website that explains software engineering domains in simple language.

## Run locally

```bash
python3 -m http.server 4174
```

Then open:

```text
http://127.0.0.1:4174
```

## Deploy on GitHub Pages

1. Create a new public GitHub repository.
2. Push these root files: `index.html`, `styles.css`, and `script.js`.
3. In the repository, open `Settings > Pages`.
4. Choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
